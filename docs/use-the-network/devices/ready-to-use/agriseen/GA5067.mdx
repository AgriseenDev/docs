## IoT based livestock GPS collar

  GA5067 is a LoRa-based (LTE cat1, LTEM optional) GPS tracking device for cattle tracking with Li-SOCl2 battery. The Collar’s GPS location data will be uplinked to the LoRaWAN gateway according to the schedule you choose, The collar is programmed wirelessly by the network server. You can control how often GPS location data are collected, and can also remotely command program changes to the fixed-rate and mortality schedule, fixed duration time, and GPS location data transmission interval.
